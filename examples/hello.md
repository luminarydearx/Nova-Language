# Hello World - Nova Language

Ini adalah program pertama dalam bahasa pemrograman **Nova v2.0**.

## Kode

```nova
// hello.nova
print("Hello, Nova v2.0!")
```

## Penjelasan

- `print()` adalah fungsi bawaan untuk menampilkan output ke konsol.
- String ditulis dengan tanda kutip ganda `"..."`.
- Komentar ditulis dengan `//` (satu baris) atau `/* ... */` (banyak baris).

## Cara Menjalankan

```bash
python main.py examples/hello.nova
```

## Output yang Diharapkan

```
Hello, Nova v2.0!
```

## Fitur yang Digunakan

- ✅ Print Statement
- ✅ String Literals
